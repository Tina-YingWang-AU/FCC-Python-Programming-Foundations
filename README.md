
# FCC-Python-Programming-Foundations

This repository contains a comprehensive suite of Python projects developed as part of the **freeCodeCamp Python Curriculum**. The collection demonstrates a progression from basic data manipulation to advanced algorithmic implementation and object-oriented architecture.

---

## 🛠️ Technical Skill Matrix

| Category | Competencies Demonstrated |
| :--- | :--- |
| **Logic & Control** | Recursive algorithms, complex conditional flow, nested iteration. |
| **Data Structures** | Manual Hash Table implementation, Stack-based logic, Dictionaries, Tuples. |
| **OOP** | Class inheritance, method overriding, encapsulation, and magic methods (`__str__`). |
| **String Engineering** | Dynamic ASCII visualization, data sanitization, and precision formatting. |

---

## 📂 Project Directory

### [01. User Configuration Manager](./01-User-Configuration-Manager/)
* **Focus**: Dictionary CRUD operations and input validation.
* **Logic**: Implements manual verification lists to ensure data integrity during runtime updates.

### [02. Budget App](./02-Budget-App/)
* **Focus**: Object-Oriented Programming and data visualization.
* **Logic**: Manages ledger-based transactions and renders a custom-scaled ASCII spending chart.

### [03. Polygon Area Calculator](./03-Polygon-Area-Calculator/)
* **Focus**: Class inheritance and the DRY (Don't Repeat Yourself) principle.
* **Logic**: Subclasses `Rectangle` into `Square`, utilizing `super()` for attribute management and method overriding for geometric consistency.

### [04. Hash Table Implementation](./04-Hash-Table-Implementation/)
* **Focus**: Computer Science fundamentals and memory mapping.
* **Logic**: Features a custom hashing function based on Unicode summation and handles collisions via nested bucket structures.

### [05. Tower of Hanoi Solver](./05-Tower-of-Hanoi-Recursive-Solver/)
* **Focus**: Recursive algorithm design and state tracking.
* **Logic**: Implements a LIFO (Last-In-First-Out) stack movement logic to solve the classic mathematical puzzle.
