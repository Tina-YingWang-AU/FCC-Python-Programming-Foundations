# ⚙️ User Configuration Manager (Project 1)

## 📝 Description
A Python utility developed for the **freeCodeCamp Python Certification**. This project focuses on managing user settings through dictionary manipulation, ensuring data integrity and case-insensitive input handling.

## 🛠️ Key Logic
* **CRUD Operations**: Implements Add, View, Update, and Delete functions.
* **Input Validation**: Uses custom logical checks to prevent duplicate settings or updating non-existent keys.
* **Data Consistency**: Ensures all configuration keys are normalized to lowercase.

## 🚀 Skills Applied
* Python Dictionaries & Tuples
* Conditional Logic (`if/else`, `all()`)
* String Formatting & Capitalization
